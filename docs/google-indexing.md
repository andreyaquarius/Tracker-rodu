# Google Search Console та індексація Трекера Роду

Цей чекліст потрібен після деплою публічних сторінок на `https://trekerrodu.com.ua`.

## 1. Відкрити Search Console

1. Перейдіть у Google Search Console.
2. Виберіть ресурс `trekerrodu.com.ua`.
3. Якщо ресурсу ще немає, додайте доменний ресурс або URL-prefix для `https://trekerrodu.com.ua/` і підтвердьте право власності способом, який пропонує Google.

## 2. Перевірити головну сторінку

1. Відкрийте інструмент URL Inspection.
2. Введіть `https://trekerrodu.com.ua/`.
3. Перевірте, що Google бачить сторінку як доступну для індексації.
4. Натисніть «Запросити індексування».

## 3. Подати sitemap

1. Перейдіть у розділ «Sitemaps».
2. Подайте sitemap:

   ```text
   https://trekerrodu.com.ua/sitemap.xml
   ```

3. Перевірте, що Search Console прийняла файл без помилок.

## 4. Перевірити публічні сторінки

Через URL Inspection окремо перевірте:

```text
https://trekerrodu.com.ua/features
https://trekerrodu.com.ua/pricing
https://trekerrodu.com.ua/privacy
https://trekerrodu.com.ua/terms
```

Для кожної сторінки перевірте, що:

- URL доступний;
- сторінку не заблоковано robots.txt;
- немає `noindex`;
- canonical збігається з перевіреним URL;
- Google бачить основний текст сторінки.

Після перевірки можна натиснути «Запросити індексування» для кожної сторінки.

## 5. Чекати індексацію

Індексація не відбувається миттєво. Після запиту Google може обробляти сторінки від кількох годин до кількох днів або довше.

Повторне щоденне подання одного й того самого URL не прискорює індексацію. Краще дочекатися оновлення статусів у Search Console.

## 6. Перевіряти розділ «Сторінки»

У розділі «Сторінки» дивіться:

- які URL проіндексовані;
- які URL не проіндексовані;
- чи немає помилок `404`;
- чи немає помилок canonical;
- чи немає випадково заблокованих сторінок.

## 7. Додаткова перевірка через Google

Пошук:

```text
site:trekerrodu.com.ua
```

можна використовувати лише як допоміжну перевірку. Він не є точним звітом про стан індексації. Основним джерелом правди є Google Search Console.

## 8. Що не потрібно подавати

Не подавайте в sitemap і URL Inspection приватні маршрути кабінету:

```text
/projects/
/settings/
/settings/subscription
```

Вони не призначені для індексації й мають бути закриті від пошукових систем.
